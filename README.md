# Student Management System

A modern Student Management System built using Flask, SQLite, HTML, and CSS.

## Features

- Add student records
- View all students
- Delete students
- Clean dashboard UI
- SQLite database integration

## Technologies Used

- Python
- Flask
- SQLite3
- HTML
- CSS

## Project Structure

student_management/
│
├── app.py
├── student.db
│
├── templates/
│   ├── index.html
│   └── students.html

I built this project while learning backend development with flask and sqlite
to strengthen my understanding of databases, routing, CRUD OPERATION, and frontend integration
